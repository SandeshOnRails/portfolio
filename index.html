VidJot

VidJot is a responsive web-app built on Node JavaScript runtime using express framework. This is a simple website for jotting down video ideas for platforms like youtube. This application uses restful routing with various HTTP requests like GET, POST, PUT(EDIT), DELETE. I developed this website while learning to use Node.js with Express, and MongoDB. I learned various aspects of this fast and light-weight web technology that is event-driven and uses non-blocking model to process requests. I also learned the usage of node modules such as passportJS to for local authentication of the user. This website uses bootstrap 4 for front-end framework, and renders amazing user experience as bootstrap 4 is mobile first front-end framework. Other various node modules used in this projects are body-parser to process form data, handlebarjs, which is a template for rendering web pages in node applications, flash-connect for flash messaging success/error reports, and bycryptjs to encrypt and decrypt passwords for users. This application is deployed on heroku, cloud-based platform, and the MongoDB databse is deployed on mLab. Please click on the below link to navigate to the web page!
